socializr
=========

Social API Scraper

A project to learn / understand:
  - Sinatra
  - CoffeeScript
  - SCSS
  - Mustache
  - Shotgun
  - HTTParty

Search in the box and it'll head off to reddit and twitter and load some stuff.
